# CLI Number Guessing Game (C Language)

A simple command-line based number guessing game built using C programming language.

## Project Description

The program generates a random number between 1 and 100.
The user keeps guessing until the correct number is found.
The system provides hints whether the guess is higher or lower than the actual number.

## Features

- Random number generation
- User input handling
- Attempt counter
- Hint system (Higher / Lower feedback)
- Loop-based game logic

## Technologies Used

- C Programming
- stdio.h
- stdlib.h
- time.h
